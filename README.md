# estimating_the_stability_figs_code
Code to produce data and figures for Estimating the stability of Labrador Sea convection with a dynamical systems approach manuscript.

There are three scripts. The first is aste_env_figs: this script uses the ASTE interface to produce Figure 1 and create the ASTE data which is used in the paper. 

The script stats_env_figs contains most of the rest of the code of interest. It contains the code to run the simulations, the code to constrain parameters and produce parameter sets, the code to create the bifurcation diagrams, and produces all of the rest of the figures. 

The script calculate_freshwater_required contains the code used to for the calculation of mass of freshwater required to cause temporary convective shutdown.
